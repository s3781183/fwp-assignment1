## Libraries Used

- Firebase - firebase is used to save the images from the post. Can store up to 10gb for free. Firebase can extract the image as a URL making it easier to retrieve and delete.
  `npm install firebase`
- Moment - moment is used to convert the time to a more readable format.
  `npm install moment`
